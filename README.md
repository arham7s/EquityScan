# EquityScan 📈

**EquityScan** is a professional-grade web application designed for real-time stock market screening and quantitative equity analysis. By bridging the gap between modern web technologies and quantitative finance principles, it empowers users to filter through market data using custom technical indicators and fundamental metrics to identify high-potential investment opportunities.

## 🚀 Live Demo
Access the live application here: [https://arham7s.github.io/EquityScan/](https://arham7s.github.io/EquityScan/)

---

## ✨ Key Features

* **Dynamic Market Screening:** Filter stocks based on customizable parameters such as Price Action, Volume, and Market Cap.
* **Quantitative Indicators:** Integrated support for technical analysis indicators including RSI, MACD, and Moving Averages (SMA/EMA).
* **Real-Time Data Integration:** Fetches up-to-date market information to ensure analysis is based on the latest trading sessions.
* **Interactive Visualizations:** Clean, responsive UI for charting and data representation, making it easier to spot trends and anomalies.
* **Performance Tracking:** Tools to evaluate equity performance across different sectors and timeframes.

## 🧠 Quantitative Logic

This tool utilizes a multi-factor ranking system to evaluate equities, focusing on:

  * **Relative Strength Index (RSI):** Identifying overbought or oversold conditions to gauge momentum.
  * **Trend Analysis:** Utilizing Moving Average crossovers (Golden Cross/Death Cross) to signal potential entry and exit points.
  * **Volatility Assessment:** Analyzing price fluctuations to determine risk-adjusted opportunities.
